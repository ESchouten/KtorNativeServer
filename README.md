# Ktor Native Server

Example how to set up a Ktor Native server using the 2.0.0 EAP releases

To start the server, run `./gradlew runReleaseExecutableNative`.

To date, Ktor Native Server (EAP 263-267) lacks support for Windows.
## Compatibility list

```
iosarm64
iossimulatorarm64
iosx64
jvm
kotlinMultiplatform

```
